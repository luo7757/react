# ImperativeHandle Hook

函数：useImperativeHandle
三个参数： 
1. 参数一 ref 对象，这个方法返回的函数会直接替换掉原本 ref 的对象
2. 参数二， 一个函数，返回的数据作为 ref.current的值
3. 参数三，依赖项，优化点，老规矩

```
function Test(props, ref){
  const Ref = useRef(ref)

  useImperativeHandle(ref, () => {
    return {
      current: Ref,
      method: () => {
        console.log('method')
      }
    }
  }, [])

  return (
    <div ref={Ref}>
      Test
    </div>
  )
}
```
